## Setup
1. Get free Groq API key from console.groq.com
2. Set up YouTube Data API v3 on console.cloud.google.com
3. Run get_refresh_token.py to get YT_REFRESH_TOKEN
4. Create GitHub repo and add 4 secrets:
   - GROQ_API_KEY
   - YT_CLIENT_ID
   - YT_CLIENT_SECRET
   - YT_REFRESH_TOKEN
5. Go to Actions and run workflow manually to test
